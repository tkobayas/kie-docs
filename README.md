# KIE (incubating) Apache Documentation repository

Documentation of KIE Apache community components.

You can find source code of the documentation for Drools, Kogito, OptaPlanner that is being published at our [website](https://kie.apache.org/docs/documentation/).

## Prerequisites

* Apache Maven
* Java

## Contribute

In order to contribute to a documentation, create an [issue](https://github.com/apache/incubator-kie-issues/), navigate to respective sub-directory, implement the changes and submit a pull-request from your fork.

Each sub-directory has its own README that describes the necesarry steps to build the documentation for the component.
